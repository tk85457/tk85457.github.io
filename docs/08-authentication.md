# SyncDub AI Product Bible

# Chapter 08 — Authentication & Onboarding

## Objective
Create a frictionless authentication flow that gets users from installation to their first translated video as quickly as possible.

## Authentication Methods
- Google Sign-In (primary)
- Email sign-in (future)

## Onboarding Flow
1. Install extension
2. Sign in with Google
3. Welcome screen
4. Grant required permissions
5. Receive free credits
6. Translate first video

## UX Principles
- Minimize required input.
- Explain permissions clearly.
- Show progress during onboarding.
- Allow users to skip non-essential steps.

## Security
- Secure OAuth flow
- Session management
- Device management
- Logout everywhere

## Success Metric
A new user should complete onboarding and start translating a video in under two minutes.
