# SyncDub AI Product Bible

# Chapter 04 — Homepage Specification

## Objective
The homepage must explain the product within seconds, build trust through interactive demonstrations, and convert visitors into extension users.

## Section Order
1. Hero
2. Interactive Demo
3. How It Works
4. Feature Highlights
5. Supported Platforms
6. Languages
7. Performance
8. Pricing Preview
9. FAQ
10. Final CTA
11. Footer

## Hero
- Clear value proposition.
- Primary CTA: Add to Chrome.
- Secondary CTA: Try Demo.
- Signature animated waveform.

## Interactive Demo
Allow visitors to compare original and translated audio with language switching and waveform feedback.

## Design Rules
- Large whitespace.
- Premium typography.
- Purposeful motion.
- Responsive from mobile to desktop.
- Accessibility-first.

## Performance
Target Lighthouse score above 95 with optimized assets and lazy loading.

## Success Metric
Visitors should understand the product, trust it, and know the next action within the first minute.
