# SyncDub AI Product Bible

# Chapter 09 — Design System

## Objective
Define a single visual language used consistently across the website, extension, dashboard, and future applications.

## Foundations
- 8px spacing system
- Consistent border radius
- Dark-first theme
- Semantic color tokens
- Variable typography scale

## Components
- Buttons
- Inputs
- Cards
- Modals
- Navigation
- Tooltips
- Toasts
- Tables
- Empty states
- Skeleton loaders

## Design Principles
- Consistency over novelty
- Accessible contrast
- Predictable interactions
- Reusable components
- Responsive by default

## Tokens
All colors, spacing, shadows, typography, and animation durations should be defined as reusable design tokens.

## Outcome
Every new screen should be buildable using existing components before introducing new patterns.
